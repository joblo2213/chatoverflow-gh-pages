GUIServlet 

Companion object GUIServlet

class GUIServlet extends HttpServlet with ScalatraServlet with WithLogger

A servlet to serve the GUI files of the chatoverflow-gui dir from the classpath. This directory is provided if the gui jar is added on the classpath. Responds with an error if the gui jar isn't on the classpath.
